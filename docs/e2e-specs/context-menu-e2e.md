# M-001: コンテキストメニュー E2Eテスト仕様書

## テスト対象
右クリックコンテキストメニュー（常前面表示切替・終了）

## テストケース

### TC-001: コンテキストメニュー表示
- **前提条件**: アプリ起動済み
- **手順**: Canvas上で右クリック
- **期待結果**: ネイティブコンテキストメニューが表示される。項目:「常に最前面に表示」「終了」

### TC-002: ブラウザデフォルトメニュー抑制
- **前提条件**: アプリ起動済み
- **手順**: Canvas上で右クリック
- **期待結果**: ブラウザ標準のコンテキストメニューは表示されない

### TC-003: 常前面表示ON
- **前提条件**: alwaysOnTop = false（デフォルト）
- **手順**: 右クリック →「常に最前面に表示」を選択
- **期待結果**: ウィンドウが最前面に固定される。チェックマークが付く

### TC-004: 常前面表示OFF
- **前提条件**: alwaysOnTop = true
- **手順**: 右クリック →「常に最前面に表示」を選択
- **期待結果**: ウィンドウの最前面固定が解除される。チェックマークが外れる

### TC-005: 設定永続化
- **前提条件**: alwaysOnTopをONに設定
- **手順**: アプリを終了 → 再起動
- **期待結果**: alwaysOnTopがONの状態で起動する

### TC-006: 終了
- **前提条件**: アプリ起動済み
- **手順**: 右クリック →「終了」を選択
- **期待結果**: アプリケーションが正常に終了する
