# 未来時計 - 要件定義書

## 1. プロジェクト概要

### 成果目標
固定された1本の針に対し、24時間表記の文字盤が反時計回りに回転することで時刻を示すデスクトップ時計アプリケーション。分・秒はデジタル数字で針上に表示する。

### 成功指標

**定量的指標:**
- 24時間表示で正確な現在時刻を反映
- 60fps以上の滑らかな文字盤回転
- 分・秒のリアルタイム更新

**定性的指標:**
- 従来にない未来的・斬新なビジュアル体験
- 文字盤の回転から直感的に時刻を読み取れる
- デスクトップに常駐させたくなる美しさ

## 3. 画面詳細仕様

### P-001: 時計画面（メイン）

**ページID:** P-001
**目的:** 未来時計の表示
**ルート:** メインウィンドウ（単一画面）

**機能:**
- Canvas上に24時間文字盤を描画
- 文字盤は反時計回りに24時間で1回転（1時間あたり15度）
- 針は12時方向（上部）に固定表示
- 針の上にデジタル数字で `MM:SS` を表示
- 現在時刻の「時」は文字盤の回転位置で示す
- 0時を基準位置とし、数字は0〜23を円周上に等間隔配置

**処理フロー:**
1. アプリ起動 → Canvas初期化（devicePixelRatio対応）
2. `requestAnimationFrame` でアニメーションループ開始
3. 毎フレーム:
   - `Date()` から現在時刻を取得
   - 時刻から文字盤の回転角度を算出: `-(hours + minutes/60) / 24 * 2π`
   - Canvas全消去 → 文字盤描画（回転適用）→ 針描画 → 分秒テキスト描画
4. `ResizeObserver` でリサイズ検知 → Canvas再設定・再描画

**UI要素:**
- 文字盤: 24個の時刻数字（0〜23）+ 目盛り
- 針: 中心から12時方向への固定線
- デジタル表示: 等幅フォントで `MM:SS` 表示
- 背景: ダークテーマ（未来的な雰囲気）

### M-001: コンテキストメニュー / トレイメニュー

## 8. 描画仕様

### 文字盤
- 24個の数字（0〜23）を円周上に等間隔（15度間隔）配置
- 0を12時方向（上部）に配置
- 数字は文字盤回転時も常に正位置（読める向き）を維持（逆回転補正必須）
- 6時間ごと（0, 6, 12, 18）に大きな目盛り

### 回転
- 反時計回り
- 24時間で360度（1回転）
- 滑らかな連続回転（ステップ動作ではない）

### 針
- 中心から12時方向に固定
- 針上にデジタル数字 `MM:SS` を表示
- 等幅フォント使用

### レスポンシブ
- `ResizeObserver` + `devicePixelRatio` でRetina対応
- ウィンドウサイズに応じて時計全体がスケーリング
